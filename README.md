### Technology

-  Nextjs
-  Material UI
-  Typescript
-  OMDB api

### How to run the app locally

```bash
# Checkout to the current branch
  git clone https://github.com/jeffubayi/whats-in

# install dependencies
  yarn 

# start the app locally
  yarn dev
```

## Features

- Search for movies 
- View movie details 
- Like favourite movie
- Mobile responsive view
